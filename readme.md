# Pricing Cards Template

A modern and responsive pricing section designed for SaaS platforms and service-based websites.  
Built with **HTML5** and **CSS3** to showcase pricing plans with a clean dark-themed UI.

---

## About

This project is a customizable pricing cards component focused on clear visual hierarchy and user engagement.  
It features multiple pricing tiers, interactive hover effects, and a highlighted “Most Popular” plan to guide user attention.

---

## Features

- Three pricing plans (Basic, Pro, Enterprise)
- Highlighted “Most Popular” plan
- Smooth hover animations and transitions
- Animated call-to-action button
- Integrated customer testimonial section
- Feature availability indicators using icons
- Fully responsive layout for all screen sizes

---

## Tech Stack

- **HTML5** – structure and semantic layout
- **CSS3** – styling, animations, and responsiveness
- **CSS Grid** – responsive card layout
- **Font Awesome** – icons
- **Google Fonts** – typography

## How to Use

1. Open `index.html` in your browser

2. Compare pricing tiers:
   - Basic
   - Pro
   - Enterprise

3. Resize the browser window to view the responsive layout

4. Customize colors by editing CSS variables in `style.css`
